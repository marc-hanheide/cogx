PCL Add Ons
===========

Requirements:
-------------
- OpenCV, PCL
- Kinect


Implementations:
----------------
- PlanePopout (with dominant plane, ROI and SOI calculation)
- PCLFunctions: 
  - FitPlanesRecursive (Plane detector)
  - ...
- PCLUtils: 
  - several functions to convert pcl to opencv formats
  - ...
