\section{Families}

\subsection{The SOMESUCH family}



To have a category with the star mode: \catg{s \backsl{\star}np \fwdsl{\star}pp}

To have a category with the dot mode:  \catg{s \backsl{\cdot}np \fwdsl{\cdot}pp}

To have a category with the diamond mode:  \catg{s \backsl{\diamond}np \fwdsl{\diamond}pp}

To typeset an unnumbered list: 

\begin{itemize}
\item here goes your item
\end{itemize}

To typeset a numbered list: 

\begin{enumerate}
\item here goes your item
\end{enumerate}
